# Carro-de-Carreras-Controlado-por-RF-Receptor-
Este proyecto corresponde al **receptor** de un **carro de carreras** controlado de forma inalámbrica mediante módulos **nRF24L01** y programado en **Arduino**.   El sistema permite controlar el movimiento del vehículo en tiempo real utilizando un **joystick**, ideal para competencias de velocidad y maniobrabilidad.
